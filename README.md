# React SPA - Práctica de Autenticación y API

Este proyecto es una aplicación de página única (SPA) desarrollada con React y Vite. Implementa navegación protegida, gestión de estado global y consumo de servicios externos.

## Estructura del Proyecto
- **pages/**: Componentes de vista (Home, Login, Dashboard, Usuarios).
- **router/**: Configuración de rutas mediante `react-router-dom` y rutas privadas.
- **services/**: Funciones para el consumo de la API REST (JSONPlaceholder).
- **store/**: Contexto de autenticación para gestionar el estado global del usuario.
